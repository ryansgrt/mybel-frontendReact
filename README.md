### Mybel-frontend-reactjs

##  How to run this project
```
1. make sure you have install nodejs and npm
2. clone mybel-backend-java in this link https://github.com/ryansgrt/mybel-API
3. run mybel-backend-java with follow the steps in readme.md
4. open cmd in this project
5. insert command: npm install
6. insert command: npm start
7. this program now run in http://localhost:3000/
8. open the browser
9. go to http://localhost:3000/
```
